Регламентные работы


Название работы Периодичность Описание работы
1. Загрузка данных в программу При необходимости загрузки новых данных о дорогах 1. Запустить программу;
2. Нажать кнопку загрузить данные;
3. Выбрать файл с данными.
2. Обновление базы данных 1 раз в месяц 1. Запустить программу;
2. Нажать кнопку просмотр данных;
3. Нажать кнопку изменить данные;
4. Изменить данные там, где это нужно.
3. Запуск программы для прогнозирования 1 раз в месяц 1. Включить компьютер;
2. Проверить подключение к интернету;
3. Запустить страницу с веб-приложением.
4. Просмотр диаграмм дорог по регионам 1 раз в месяц 1. Запустить программу;
2. Нажать кнопку диаграммы;
3. Увидеть диаграммы сети дорог по регионам.
